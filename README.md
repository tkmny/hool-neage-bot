# 価格改定・値上げ速報 自動配信システム

食品・光熱費・通信費など全業種の価格改定情報を
毎日自動収集・生成・配信するHuman Out of the Loopシステム。

## システムパイプライン

```
PR TIMES RSS → Claude API（2文体生成）→ Threads Graph API（自動投稿）
                                         ↓
                          週次エンゲージメント比較 → プロンプト自動更新
```

## 3軸評価

| 軸 | 評価 | 内容 |
|---|---|---|
| 稼働性 | ◎ | 完全自動パイプライン。人間介入ゼロ。 |
| 進化性 | ◎ | 2文体競争→週次で勝ち文体をプロンプトに自動採用。毎週進化する。 |
| 事業性 | ○ | フォロワー増加→プロフィールLINE登録→LINE OA誘導。 |
